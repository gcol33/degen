# Compute profile likelihood for one parameter

Compute profile likelihood for one parameter

## Usage

``` r
profile_likelihood(spec, y, par, which_par, n_points = 20)
```

## Arguments

- spec:

  A model_spec object

- y:

  Data

- par:

  Starting parameter values

- which_par:

  Name of parameter to profile

- n_points:

  Number of points in profile

## Value

Data frame with profile likelihood values
